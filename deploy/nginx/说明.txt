开始安装nginx。
 sudo netstat -tunlp | grep 80 查看端口
1、直接拉取最新的nginx镜像

docker pull nginx

2、新建一些目录，把nginx容器内的相关文件夹挂载到宿主机上，主要方便改写配置，其次是删除容器，这些文件不会丢失

mkdir -p /etc/docker/nginx/{conf,html,logs,certs}

3、启动一个nginx容器，用来cp一些文件，放到步骤2新建的文件夹里面

docker run --name nginx -p 80:80 -d nginx

4、进入nginx 容器 查看复制文件
docker exec -it nginx /bin/bash

whereis nginx
exit 
docker cp nginx:/etc/nginx/conf.d/default.conf ./conf/
docker cp nginx:/etc/nginx/nginx.conf ./
docker stop nginx
docker remove nginx

7、compose up 
docker compose -f docker-compose-nginx.yml up -d
docker compose -f docker-compose-nginx.yml restart
